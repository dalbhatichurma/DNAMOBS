### 1. Build the Plugin
```batch
cd C:\DNAMobs

```

### 2. Install on Server
```batch
copy target\DNAMobs-1.0.0.jar <server>\plugins\
```

The plugin will auto-generate config files

### 4. Configure
Edit `plugins/DNAMobs/config.yml`

### 5. Test
Use `/dna stats` and kill some mobs
